📖 Gmail App Password Setup Guide

2. Go to Security → enable 2-Step Verification if not already on
3. Search "App passwords" in the search bar at the top
4. App: Mail · Device: Windows Computer → click Generate
5. Copy the 16-character password (no spaces) → paste into App Password field above
6. Enter the same Gmail address you generated the app password for
7. Click Save Settings → Send Test Email to confirm it works
